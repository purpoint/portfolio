DROP YOUR PROJECT SCREENSHOTS IN THIS FOLDER
============================================

The "Selected Work" section shows one card per project. Each card has space
for a picture of the project. Put the pictures here, named EXACTLY like this:

    collabboard.png     ->  CollabBoard  (https://collab-board-xi.vercel.app)
    wasteiq.png         ->  WasteIQ      (https://waste-iq-plum.vercel.app)
    mindvault.png       ->  MindVault    (https://mindvault-pink.vercel.app)
    echovision.png      ->  EchoVision AI

HOW:
  1. Open each project's live site in your browser.
  2. Take a screenshot (Mac: Cmd + Shift + 4, then drag to select).
  3. Rename the file to match the names above.
  4. Drag it into this folder (manan_portflio/public/projects/).
  5. Refresh the portfolio — the picture appears automatically, with a warm
     tint that turns to full colour when you hover.

NOTES:
  - Landscape shots (roughly 16:11, e.g. 1280x880) look best.
  - .png is what the code expects. If you use .jpg instead, also change the
    "image" path for that project in src/data.ts.
  - You can delete this README.txt once you've added your screenshots.
